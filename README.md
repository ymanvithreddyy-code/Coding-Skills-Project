# Student Record Management System (SRMS)

A comprehensive web-based application for managing student records, academic performance, and administrative requests. This system provides distinct portals for Administrators and Students, ensuring secure and efficient data management.

### Admin Portal
*   **Dashboard:** Visual overview of student statistics, attendance, and performance.
*   **Student Management:** Add, Edit, Delete, and View student profiles.
*   **Academic Management:** Manage Courses, Marks, and Attendance records.
*   **Request Management:** Review and action (Approve/Reject/Resolve) student requests and complaints.

### Student Portal
*   **Personal Dashboard:** View attendance percentage, average marks, and active courses.
*   **Profile View:** Access personal and academic details.
*   **Academic Records:** View obtained marks, grades, and attendance history.
*   **Request System:** Raise tickets for corrections or complaints and track their status.
